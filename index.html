Animations

Data

First, the data from the existing graph must be updated. In order to do this we need to:
  1. Modify the values in our dataset
  2. Rebind the new values to the existing elements and overwrite the original data values.

    This is done by selecting our rects and calling .data() again

  3. Set new attribute values to update the visual display

    This is done by copying and pasting the scale code we've already written for the original graph. Additionally, the data labels and rect colors must be updates as well.

All of this is done in a new funciton that contains an on("click") parameter

Axis

Once we have our data we need to update the axes. For both the x and y (for a scatterplot at least), we need to:
  1. Select the axis
  2. Initiate a transiton
  3. Set the transition's duration
  4. Call the approptiate axis generator

Transitions

Smooth transitions can also be added by .transition() and the duration (how long the transition will take) of the transition can be specified. However it is important to note in our code that the duration must come after the transition.

Once we have a transition, we can make something happen at the beginning or end of a transition using .on() in conjunction with a new function to execute some type of action.
